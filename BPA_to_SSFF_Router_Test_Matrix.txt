============================================

  If SuccessFactors returns lowercase 'error', condition will not trigger
  
Affected Route:       Route 1 (Error route)

Recommendation:

  upper-case(/UpsertResponses/PositionUpsertResponse/status) = 'ERROR'


ISSUE 2: NO SCHEMA VALIDATION (CRITICAL SEVERITY)
===================================================
Test Cases:           TC_ROUTER_1_4, TC_ROUTER_1_5, TC_ROUTER_2_4
Description:
  Router assumes response conforms to expected XML schema:
  <UpsertResponses><PositionUpsertResponse><status>...
  
  If API returns different structure, XPath returns empty sequence
